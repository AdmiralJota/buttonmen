General info about Button Men

Button Men was created by Cheapass Games, who have graciously given us their blessing to run this free and non-commercial site.

Longtime online Button Men player Spindisc has a site with lots of general info about the game.

Developer info

This site has a project on GitHub, and you can browse the repository there.

If you'd like to contribute to the project, check out the developer guide. You can also view and create tickets about bugs, feature requests, etc. Or just post to the forums if you don't have (and don't want) a GitHub account.
